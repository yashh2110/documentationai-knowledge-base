## Quick Start

Install the Paperguide SDK to integrate into your apps.

<CodeGroup show-lines="true" tabs={[]}>
  ```bash
  npm install @paperguide/sdk
  ```

  ```bash
  yarn add @paperguide/sdk
  ```

  ```bash
  pnpm add @paperguide/sdk
  ```
</CodeGroup>

<Steps>
  <Step title="Initialize SDK" icon="code" title-type="p">
    ```javascript
    import { Paperguide } from '@paperguide/sdk';

    const pg = new Paperguide({ apiKey: 'YOUR_API_KEY' });
    ```
  </Step>

  <Step title="Search Papers" icon="search" title-type="p">
    ```javascript
    const results = await pg.search('machine learning ethics');
    console.log(results.papers);
    ```
  </Step>

  <Step title="Analyze Document" icon="zap" title-type="p">
    ```javascript
    const summary = await pg.analyze('path/to/paper.pdf');
    console.log(summary.keyInsights);
    ```
  </Step>
</Steps>
